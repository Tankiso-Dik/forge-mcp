# Answers

## Group A

1. \(\frac{3}{4} + \frac{5}{6} - \frac{7}{12} = \frac{9}{12} + \frac{10}{12} - \frac{7}{12} = \frac{12}{12} = 1\)
2. \(18^2 - 17^2 = (18 - 17)(18 + 17) = 1 \cdot 35 = 35\)
3. \(\frac{2}{3} \times \frac{9}{14} \div \frac{3}{7} = \frac{2}{3} \times \frac{9}{14} \times \frac{7}{3} = \frac{3}{7} \times \frac{7}{3} = 1\)
4. \(\gcd(252, 198)\): \(252 - 198 = 54\), \(198 \bmod 54 = 36\), \(54 \bmod 36 = 18\), \(36 \bmod 18 = 0\), so the gcd is \(18\)

## Group B

1. \(3x - 7 = 2x + 11 \Rightarrow x - 7 = 11 \Rightarrow x = 18\)
2. \(x^2 - 9x + 20 = 0 = (x - 4)(x - 5)\), so \(x = 4\) or \(x = 5\)
3. From \(a - b = 2\), \(a = b + 2\). Substitute into \(2a + 3b = 19\):
   \(2(b + 2) + 3b = 19 \Rightarrow 5b + 4 = 19 \Rightarrow 5b = 15 \Rightarrow b = 3\), so \(a = 5\)
