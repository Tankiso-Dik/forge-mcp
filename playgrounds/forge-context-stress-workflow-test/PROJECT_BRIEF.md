# Project Brief

Project codename: `Northstar Desk`

Northstar Desk is a personal planning assistant for students who manage:
- assignments
- reading tasks
- deadlines
- weekly routines
- self-check reviews

The intended first release is deliberately narrow:
- one primary user
- local-first behavior
- low setup friction
- easy recovery after interruptions

What the product should feel like:
- calm
- compact
- trustworthy
- not overloaded with dashboards

What matters most:
1. users should quickly see what matters this week
2. missed work should be easy to recover from
3. planning should not require constant manual re-entry

Known non-goals for the first release:
- multi-user collaboration
- cloud-only operation
- advanced analytics
- heavy notification systems

Initial deliverables for this workspace:
- a coherent product specification
- a plausible architecture plan
- an explicit risk register
- a short executive summary
