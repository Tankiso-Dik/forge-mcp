# Rollout Notes

Proposed rollout concept:

## Milestone 1

- local data model
- weekly view
- task and routine capture
- recovery flow

## Milestone 2

- calendar export
- improved review history
- usability polish based on early feedback

## Milestone 3

- optional import/export improvements
- limited portability between devices

## Known rollout concerns

- If the first release scope spreads too far, the local-first advantage gets diluted.
- If recovery flow is weak, the product loses its main differentiator.
- If the system promises too much synchronization too early, trust may drop fast.
