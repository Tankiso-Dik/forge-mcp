## Set B

1. \(\gcd(420, 378)\): \(420 - 378 = 42\), and \(378 \bmod 42 = 0\), so the gcd is \(42\)
2. From \(m + n = 8\), \(n = 8 - m\). Substitute into \(2m - n = 7\):
   \(2m - (8 - m) = 7 \Rightarrow 3m - 8 = 7 \Rightarrow 3m = 15 \Rightarrow m = 5\), so \(n = 3\)
