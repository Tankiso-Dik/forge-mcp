# Longitudinal Reasoning Tasks

The work is intentionally split across multiple sessions.

## Session 1: Foundations

Solve these and add concise derivations to `WORKBOOK.md`.

### Set A

1. Compute:
   \[
   \frac{5}{8} + \frac{7}{12} - \frac{1}{3}
   \]

2. Compute:
   \[
   29^2 - 21^2
   \]

3. Solve for `x`:
   \[
   4x + 9 = 3x + 23
   \]

4. Solve:
   \[
   x^2 - 11x + 24 = 0
   \]

### Set B

1. Find the greatest common divisor of `420` and `378`.

2. If `2m - n = 7` and `m + n = 8`, find `m` and `n`.

## Session 2: Verify And Extend

First review Session 1 work for correctness and clarity.

Then solve these and continue updating `WORKBOOK.md`.

### Set C

1. How many integers from `1` to `150` are divisible by `5` or `9`?

2. A number is chosen from `1` to `120`. What is the probability it is divisible by `4` but not by `6`? Give the reduced fraction.

3. Find the sum of the first `15` positive integers.

4. Find the sum of the arithmetic sequence:
   `7 + 11 + 15 + ... + 63`

At the end of Session 2, extract reusable solving patterns into `HEURISTICS.md`.

## Session 3: Deliverable Change

A new requirement arrives:

- the raw workbook is no longer the only deliverable
- create a beginner-friendly `STUDY_GUIDE.md`
- include:
  - a short “how to approach these problems” section
  - a compact summary of the most useful solving patterns
  - 5 self-check questions
  - a short answer key

You may reuse prior work, but the guide should feel intentionally written for a learner, not copied blindly from the workbook.

## Session 4: Final Review

Review the whole workspace and finish cleanly.

Check for:
- correctness
- consistency between files
- unnecessary duplication
- weak explanations
- stale phase or issue state

Then close out the remaining work and leave a final handoff.
